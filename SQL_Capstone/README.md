# SQL Capstone

## Overview

This folder contains my SQL capstone project completed during the CyberSafe Foundation DigiGirls Data Analysis Program.

The project involved answering SQL questions using PostgreSQL and the dvdrental sample database.

## Project Objectives

- Answer business questions using SQL.
- Practise writing SQL queries to retrieve and analyze data.
- Work with data stored in multiple related tables.
- Strengthen my understanding of SQL concepts through practical exercises.

## Skills Practised

- Writing SQL queries
- Filtering and sorting data
- Joining multiple tables
- Using aggregate functions
- Grouping data with `GROUP BY`
- Ordering query results
- Limiting query results

## Files

- **SQL_Queries.sql** – My SQL solutions to the project questions.
- **Project_Questions.docx** – The original project questions.
- **Screenshots** – Screenshots of my original SQL query results.

## What I Learned

Through this project, I gained practical experience writing SQL queries to answer business questions using data from multiple related tables.
