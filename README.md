# DigiGirls Data Analysis Training

This repository documents my learning and capstone project from the CyberSafe Foundation DigiGirls Data Analysis Program, completed in 2023.

## About the Program

The program included training in:

- Basic Digital Literacy
- Cybersecurity Awareness
- Soft Skills
- Data Analytics Training (8 weeks)

## Repository Contents

This repository includes:

- Excel Capstone Project
- SQL Capstone Project
- Certificate of Completion

This repository captures the beginning of my data analytics journey and the practical skills I gained during the program.
