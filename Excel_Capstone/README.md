# Excel Capstone Project

## Overview

This project was completed as part of the CyberSafe Foundation DigiGirls Data Analysis Program.

The project required using Microsoft Excel to analyze a sales dataset and answer business questions using PivotTables and charts. It also involved interpreting the results and making recommendations based on the analysis.

## Project Objectives

The project required analyzing a sales dataset to answer business questions, including:

- Identifying the best and least performing regions and countries based on revenue and profit.
- Analyzing sales performance across different years.
- Comparing sales performance across regions, countries, product categories, and sales channels.
- Creating charts to present trends and comparisons.
- Making business recommendations based on the analysis.

## Data Preparation

To support the analysis, I extracted the **Order Year** from the **Order Date** column. This additional column made it easier to analyze the data across different years and answer several of the project questions.

## Excel Skills Practised

During this project, I practised using:

- PivotTables
- Charts
- Sorting and filtering
- Data preparation
- Business data analysis

This project helped me build confidence in using Microsoft Excel to analyze data and answer business questions. I also learned the importance of preparing data before analysis, using PivotTables and charts to summarize information, and presenting findings in a clear and meaningful way.
